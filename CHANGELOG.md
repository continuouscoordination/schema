# Changelog

All notable changes to the Continuous Coordination Schema will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-09

- Person becomes actor to reflect autonomous agents as part of the coordination loops.
- Remove `next_steps` from goal story updates. Intent for goal stories is captured in the goal story itself.

## [0.2.0] - 2026-03-08

Remove `email` from person schema; contact info is an implementation detail.

## [0.1.0] - 2026-03-05

Initial release.
